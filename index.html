📝 입력 시트

이 시스템을 어디에 올려서 사용하면 될까? G드라이브나 어디에 올리면 웹페이지로 올릴 수 있어?

이 시스템을 사용하는 가장 간단한 방법들을 알려드릴게요:

🌟 추천 방법

1. HTML 파일로 저장해서 사용 (가장 간단!)

지금 보시는 화면 오른쪽 상단의 다운로드 버튼을 눌러 HTML 파일로 저장하신 후:

  • 컴퓨터 아무 폴더에나 저장
  • 파일을 더블클릭하면 웹브라우저에서 바로 실행
  • 북마크에 추가해두면 언제든지 접속 가능
  • 인터넷 연결 없이도 사용 가능!

2. 무료 웹호스팅 서비스 이용

  • GitHub Pages (추천)
    • 무료, 광고 없음
    • 간단한 설정으로 yourname.github.io 같은 주소 생성
  • Netlify 또는 Vercel
    • 무료 플랜 제공
    • 파일 드래그 앤 드롭만으로 배포 완료

3. 구글 드라이브는?

구글 드라이브에 HTML 파일을 올려도 웹페이지로 직접 실행되지 않습니다. 드라이브는 파일 저장용이라 웹호스팅 기능이 없어요.

📝 GitHub Pages 배포 가이드

1단계: 파일 다운로드

위의 HTML 파일을 index.html 이름으로 저장하세요.

2단계: GitHub 저장소 만들기

  1. GitHub.com 접속 후 로그인
  2. 오른쪽 상단 "+" 버튼 → "New repository" 클릭
  3. 저장소 설정:
    • Repository name: caregiver-schedule (원하는 이름)
    • Public 선택
    • "Add a README file" 체크
    • Create repository 클릭

3단계: 파일 업로드

  1. 저장소 페이지에서 "Add file" → "Upload files" 클릭
  2. 다운로드한 index.html 파일을 드래그 앤 드롭
  3. "Commit changes" 버튼 클릭

4단계: GitHub Pages 활성화

  1. 저장소 상단 메뉴에서 "Settings" 클릭
  2. 왼쪽 메뉴에서 "Pages" 클릭
  3. Source 섹션에서:
    • Branch: main 선택
    • 폴더: / (root) 선택
    • Save 클릭

5단계: 완료!

약 1~2분 후 페이지 상단에 주소가 나타납니다:

https://your-username.github.io/caregiver-schedule/
